## Witch 是什么

Witch 是 **Many Tricks** 工作室出品的 Mac 窗口切换工具，已经迭代到 4.7 版本，支持 macOS 10.13 High Sierra 及以上系统，包括最新的 Apple Silicon Mac。

它的核心逻辑很简单：**按下快捷键，弹出一个列表，列表里是你所有打开的窗口和标签页，你直接选，直接跳。**

不用先切到应用再找窗口，不用鼠标乱点，直达目标。

## 说说具体能干什么

**精准切窗口，不是切应用**

按下 Option+Tab，Witch 面板出来了。里面列的不是"Chrome、Figma、Notion"这种应用图标——而是"Chrome 第一个窗口（标题：GitHub）、Chrome 第二个窗口（标题：某文档）、Figma 某设计稿……"。你一眼看清楚，点或者继续按 Tab 定位，松手，直接过去。

这感觉有点像从前 Windows 用 Alt+Tab 能直接跳到具体窗口——很多从 Windows 转过来的用户第一次用 Witch 会觉得"终于找到组织了"。

**模糊搜索，窗口多也不怕**

面板弹出来之后直接打字，Witch 实时过滤匹配，窗口列表立刻缩短到你想要的那几个。对同时维护多个项目的开发者、设计师来说，这个功能每天能省不少来回找窗口的时间。

**菜单栏模式，不习惯快捷键也行**

如果你不想记快捷键，还有另一条路：把 Witch 设成菜单栏图标，点开下拉菜单，所有窗口在那里，一眼全看到，点一下就切过去。

**多套切换方案，随便折腾**

Witch 允许你设置多套切换方案，每套方案独立配置快捷键、排序方式、窗口类型（只看最小化的？只看当前 Space 的？），完全按照自己的工作流来。外观也可以自定义：面板大小、配色、字体字号，怎么顺眼怎么来。

**排除列表，避免热键冲突**

某些应用里你不想让 Witch 接管快捷键？加进排除列表就好了，在那个应用里 Witch 老老实实不弹出来。

## 和 macOS 自带切换器比，差距在哪

说实话，对于只开两三个应用、每个应用一个窗口的用户来说，系统自带的完全够用。

但一旦你的工作台开始复杂起来——多个浏览器窗口、多个 Terminal 会话、多个 Figma 文件、多个 Finder 窗口——自带切换器就开始掉链子了。它看不见独立的窗口，只能切应用；它不支持标签页切换；它的列表也不能搜索。

Witch 在这些地方全部补上了。
